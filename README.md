# 电商用户行为分析与购买预测



## 项目介绍:

这是一个完整的电商数据分析项目，基于真实数据集，包含数据清洗、探索分析、客户分群、机器学习建模和业务洞察。



##### 快速开始

1\. 安装依赖：`pip install -r requirements.txt`

2\.打开 `电商数据分析.ipynb`

3\. 运行代码块



##### 项目流程

1\. 数据加载与清洗

2\.探索性数据分析

3\.RFM客户分群

4\.购买预测建模

5\.业务建议



##### 技术栈

Python, Pandas, Scikit-learn,RandomForest，XGBoost, Matplotlib
